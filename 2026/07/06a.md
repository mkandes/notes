### 3. Benchmark Job Scripts
- If you have not done so already, once you've sorted out how to run your benchmark, you'll then want to codify everything you've learned about how to run the benchmark by creating a batch jon script. Here is my example benchmark job script for ResNet-50:

```
#!/usr/bin/env bash
#
# Run MLPerf Inference: Datacenter Benchmark for Image Classification

#SBATCH --job-name=mlperf-inference-datacenter-vision-image-classification-resnet-imagenet-onnxruntime-1.17.3-cuda-a100-pcie-40g-offline-performance
#SBATCH --account=use300
#SBATCH --reservation=path-testing
#SBATCH --clusters=expanse
#SBATCH --partition=gpu-shared
#SBATCH --nodes=1
#SBATCH --ntasks-per-node=1
#SBATCH --cpus-per-task=16
#SBATCH --mem=96G
#SBATCH --gpus=1
#SBATCH --time=00:30:00
#SBATCH --output=%x.o%j.%N
#SBATCH --array=0
#SBATCH --exclude=exp-17-57

declare -xr DATETIME_UNIX="$(date +'%s')"
declare -xr DATETIME_UTC="$(date -u +'%Y%m%dT%H%M%S%z')"
declare -xr DATETIME_LTZ="$(date +'%Y%m%dT%H%M%S%z')"

declare -xr SLURM_JOB_SCRIPT="$(scontrol show job ${SLURM_JOB_ID} | awk -F= '/Command=/{print $2}')"
declare -xr SLURM_JOB_SCRIPT_MD5="$(md5sum ${SLURM_JOB_SCRIPT} | awk '{print $1}')"
declare -xr SLURM_JOB_SCRIPT_SHA256="$(sha256sum ${SLURM_JOB_SCRIPT} | awk '{print $1}')"
declare -xr SLURM_JOB_SCRIPT_LINES="$(wc -l ${SLURM_JOB_SCRIPT} | awk '{print $1}')"

declare -xr LUSTRE_PROJECT_DIR="/expanse/lustre/projects/${SLURM_ACCOUNT}/${USER}"
declare -xr LUSTRE_SCRATCH_DIR="/expanse/lustre/scratch/${USER}/temp_project"

declare -xr LOCAL_SCRATCH_DIR="${SLURM_TMPDIR}"

declare -xr CONDA_ROOT="${LOCAL_SCRATCH_DIR}/miniforge"
declare -xr CONDA_ENVS_PATH="${CONDA_ROOT}/envs"
declare -xr CONDA_PKGS_DIRS="${CONDA_ROOT}/pkgs"

declare -xr CONDA_ENV_YAML="${SLURM_SUBMIT_DIR}/mlperf-inference-datacenter-vision-image-classification-resnet-imagenet-onnxruntime-1.17.3-cuda.yaml"
declare -xr CONDA_ENV_NAME="$(grep '^name:' ${CONDA_ENV_YAML} | awk '{print $2}')"

declare -xr MODEL_DIR="${LOCAL_SCRATCH_DIR}"
declare -xr DATA_DIR="${LOCAL_SCRATCH_DIR}/ILSVRC2012_img_val"

echo "${DATETIME_UNIX} ${DATETIME_UTC} ${DATETIME_LTZ} ${SLURM_JOB_ID} ${SLURM_JOB_SCRIPT_MD5} ${SLURM_JOB_SCRIPT_SHA256} ${SLURM_JOB_SCRIPT_LINES} ${SLURM_JOB_SCRIPT}"
cat "${SLURM_JOB_SCRIPT}"

module reset
module list
#export CUDA_VISIBLE_DEVICES="$((RANDOM%4))"
export ORT_DISABLE_THREAD_AFFINITY=1
printenv

cd "${LOCAL_SCRATCH_DIR}"

wget https://github.com/conda-forge/miniforge/releases/latest/download/Miniforge3-Linux-x86_64.sh
bash Miniforge3-Linux-x86_64.sh -b -p "${CONDA_ROOT}"
source "${CONDA_ROOT}/etc/profile.d/conda.sh"
conda env create --file "${CONDA_ENV_YAML}"
conda activate "${CONDA_ENV_NAME}"
conda env export > "${SLURM_JOB_NAME}-${SLURM_JOB_ID}.yaml"

# opset-8/11
#wget https://zenodo.org/record/2592612/files/resnet50_v1.onnx
wget https://zenodo.org/record/4735647/files/resnet50_v1.onnx

mkdir -p "${DATA_DIR}"
time -p wget https://www.image-net.org/data/ILSVRC/2012/ILSVRC2012_img_val.tar
time -p tar -xf ILSVRC2012_img_val.tar -C "${DATA_DIR}"
cp "${HOME}/data/imagenet/ilsvrc/2012/val_map.txt" "${DATA_DIR}"

git clone https://github.com/mlcommons/inference.git
export PYTHONPATH="${LOCAL_SCRATCH_DIR}/inference/vision/classification_and_detection/python:${PYTHONPATH}"
cd inference/vision/classification_and_detection
time -p srun -n 1 python3 python/main.py \
  --profile resnet50-onnxruntime \
  --model "${MODEL_DIR}/resnet50_v1.onnx" \
  --dataset-path "${DATA_DIR}" \
  --output "${SLURM_SUBMIT_DIR}/results/a100-pcie-40g/onnxruntime-gpu/resnet50/offline/performance/${SLURM_ARRAY_JOB_ID}/${SLURM_ARRAY_TASK_ID}" \
  --scenario Offline \
  --max-batchsize 256 \
  --threads 4 \
  --qps 5120
```
