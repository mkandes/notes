# How to setup and run [Doccano](https://doccano.github.io/doccano) locally on your personal computer

1. [Download and install miniforge](https://conda-forge.org/download) on your personal computer.
   
2. Copy and paste the following conda-forge environment into a text file named `doccano.yaml`.

```
name: doccano

channels:
  - conda-forge

dependencies:
  - python=3.8
  - pip
  - pip:
    - doccano
```

3. Then use the `doccano.yaml` environment file to create a new conda environment.

*Command*
```
conda env create -f doccano.yaml
```

4. Once the installation is complete, activate the `doccano` environment.

*Command*
```
conda activate doccano
```

*Output*
```
mkandes@hardtack:~/projects/rehs/2026/llm$ conda activate doccano
(doccano) mkandes@hardtack:~/projects/rehs/2026/llm$
```

5. Then initialize its database.

*Command*
```
doccano init
```
