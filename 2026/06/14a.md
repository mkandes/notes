# How to setup and run [Doccano](https://doccano.github.io/doccano) locally on your personal computer

1. [Download and install miniforge](https://conda-forge.org/download) on your personal computer.
   
2. Create a new conda environment using the following YAML file.

`doccano.yaml`
```
name: doccano

channels:
  - conda-forge

dependencies:
  - python=3.8
  - pip
  - pip:
    - doccano
```

3. ...
