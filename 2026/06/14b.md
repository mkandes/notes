# How to setup and run [Label Studio](https://labelstud.io) locally on your personal computer

### 1. [Download and install miniforge](https://conda-forge.org/download) on your personal computer.
   
### 2. Copy and paste the following conda-forge environment into a text file named `label-studio.yaml`.

```
name: label-studio

channels:
  - conda-forge

dependencies:
  - python=3.9
  - pip
  - pip:
    - label-studio
```

### 3. Then use the `label-studio.yaml` environment file to create a new conda environment.

*Command*
```
conda env create -f label-studio.yaml
```
