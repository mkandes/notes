### Developing software environments with conda environment files and galyleo on Expanse

1. Login to Expanse either via SSH or the [Expanse User Portal](https://portal.expanse.sdsc.edu).
2. If this is your first time running galyleo, put in in your PATH environemnt variable and set it in your .bashrc file.
```
echo 'export PATH="/cm/shared/apps/sdsc/galyleo:${PATH}"' >> .bashrc
```
3. Create a conda environemnt YAML file in your HOME directory on Expanse and copy and paste the following environment.

```
name: notebooks-sharing

channels:
  - conda-forge

dependencies:
  - python
  - jupyterlab
  - pandas
  - matplotlib
  - seaborn
  - scikit-learn
```

4. You can then launch your JupyterLab session.

```
galyleo launch --account ddp324 --partition debug --cpus 4 --memory 8 --time-limit 00:30:00 --conda-yml example.yaml
```

5. To access your JupyterLab session, copy and paste the URL provided by galyleo.

```
https://passerby-siding-skimmer.expanse-user-content.sdsc.edu/?token=XXXXXXXXXXXXXXXXXXXXXX
```
